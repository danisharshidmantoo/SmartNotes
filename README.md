# SmartNotes

AI-powered note-taking web application.

## Features
- Create and manage notes
- AI-based summarization (Gemini API)
- User authentication
- Search functionality

## Tech Stack
- Frontend: React
- Backend: Node.js, Express
- Database: MongoDB
- AI: Gemini API

## Setup

```bash
npm install
npm start
